# Scripts

Automation and utility scripts for Budget Reduction Tracking.

## Agent Responsible

- **Agent 1**: Infrastructure & DevOps Engineer

## Scripts

- `dev-setup.sh` - Initial development environment setup
- `backup-db.sh` - Database backup script
- `restore-db.sh` - Database restore script
- `deploy.sh` - Deployment automation
- `health-check.sh` - Application health check

## Usage

```bash
# Make scripts executable
chmod +x scripts/*.sh

# Run setup
./scripts/dev-setup.sh
```
